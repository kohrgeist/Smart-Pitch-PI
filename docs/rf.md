# Requisitos Funcionais

### RF01 — Login do Usuário
O sistema deve permitir que o usuário realize login com e-mail e senha.

### RF02 — Recuperar Senha
O sistema deve permitir que o usuário solicite recuperação de senha por e-mail.

### RF03 — Atualização de Perfil
O usuário pode alterar email e nome.

### RF04 — Cadastro de Novos Usuários
O sistema deve permitir que novos usuários sejam criados. 

### RF05 — Entrada de URLs Para Pesquisa
O sistema deve permitir que o usuário usuário insira a URL do perfil do Linkedin do indivíduo, assim como a URL do site da empresa.

### RF06 — Atualização do Processo
O sistema deve apresentar a atualização do processo de pesquisa ao usuário.

### RF07 — Geração de Scripts
O sistema deve utilizar dados extraídos da pesquisa para gerar, junto à IA, roteiros de e-mail e chamada personalizados para o setor comercial.

### RF08 — Apresentação dos Modelos Gerados
O sistema deve apresentar ao usuário os modelos de script gerados.

### RF09 — Gerar Novo Modelo
O sistema deve apresentar a opção para o usuário requisitar que um novo modelo seja gerado a partir da pesquisa já realizada.

### RF10 — Histórico
O sistema deve salvar o histórico de todas as pesquisas realizadas pelo usuário nos últimos sete dias.

### RF11 — Deletar Histórico
O sistema pode deletar histórico de pesquisa que exceda o tempo de sete dias.

### RF12 — Favoritar Scripts
O sistema deve permitir que o usuário marque os roteiros como "Favorito" para salvar.

### RF13 — Lista de Favoritos 
O sistema deve apresentar ao usuário a lista de Favoritos ao usuário.
