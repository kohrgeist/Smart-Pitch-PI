## Regras de Negócio (RN)

### RN01 — Maioridade
Para criar uma conta, o usuário deve ter mais de 18 anos.

### RN02 — Bloqueio por inatividade
Usuários inativos por mais de 90 dias devem ser marcados como inativos.

### RN03 – Bloqueio de login
Após errar 5 vezes a senha ou e-mail, bloquear tentativas pro 5 minutos.

### RN04 – Bloquear pesquisas sem um link digitado.
O botão de pesquisa só deve ser habilitado se ambos os campos estiverem preenchidos.

### RN05 – Validar formato do link.
O sistema só deve processar URLs do linkedin.

### RN06 – Limite de histórico.
O histórico deve armazenar apenas as últimas 50 pesquisas realizadas pelo usuário.

### RN07 – Igualdade de nova senha.
Na redefinição, a alteração só será salva se o campo "Nova Senha" for idêntico ao campo "Confirmar nova Senha".
